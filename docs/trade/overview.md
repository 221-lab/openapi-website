### 交易接口总览

| 模块 | 功能     | 接口名        | 功能简介                   |
| ---- | -------- | ------------- | -------------------------- |
| 订单 | 下单     | Submit        | 港美股，窝轮，期权委托下单 |
| 订单 | 改单     | Replace       | 修改订单的价格和数量等     |
| 订单 | 撤单     | Withdraw      | 订单撤单                   |
| 订单 | 当日订单 | TodayOrders   | 获取当日订单和订单详情     |
| 订单 | 历史订单 | HistoryOrders | 获取历史订单               |
| 订单 | 当日成交 | TodayTrades   | 获取当日成交               |
| 订单 | 历史成交 | HistoryTrades | 获取历史成交               |
